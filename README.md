# Simple Node.js Express Backend

This is a minimal Express.js backend application.

## How to Run

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   npm start
   ```
3. Visit [http://localhost:3000](http://localhost:3000) to see the response.
